# Threader 🧵

Autonomous agentic system for Meta Threads marketing.

## Features
- **Project Namespaces**: Register multiple products/projects.
- **Infinite Media Indexing**: Upload and index media using AI & Tesseract OCR.
- **Human-Sounding Posts**: AI-crafted content tailored to brand voice.
- **Leads & Engagement**: Automated marketing and lead acquisition.

## Setup
```bash
go mod download
go build ./cmd/threader
./threader
```

## Integration
Threader plugs into the **Vibe Auracle** provider for AI capabilities and secure credential storage.
